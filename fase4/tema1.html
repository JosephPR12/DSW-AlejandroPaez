Flexbox

Uso de flexbox: Usado en Hojas de estilo en cascada CSS; es creado para mejorar la forma en que se hace un diseño responsivo del sitio evitando el continuo uso de float, reduciendo la cantidad de lineas de codigo y facilitiando el posicionamiento de los elementos (alterando, alto y distribución de elementos), optimiza el espacio disponible.

La estructura Flexbox, se compone de contendores padre e hijos: El contenedor-Flex es el contenedor padre, contiene las direcciones que definen origen y fin del flujo de elementos hijos que se desean posicionar El elemento-flex es el elemento hijo, útiliza el espacio disponible para ubicarse de acuerdo a las propiedades y ejes (horizontal y vertical) a los cuales está sometido.

Propiedades: Para usar Flexbox se debe iniciar definiendo la propiedad "display: flex" en el contenedor padre:

  • flex-direction: define el eje principal y secundario de elemento hijo, puede ser: row, row-reverse, column, column-reverse
  • flex-wrap: por defecto flex, dispone los elementos en una misma línea, pero si se usa wrap se puede ordenar los elementos en más de una fila o columna; puede ser: nowrap, wrap, wrap-reverse
  • justify-content: da direccionamiento horizontal a los elementos hijos; puede ser: strech, flex-start, flex-end, center, space-between, space-around y space-evenly
  • align-items: da direccionamiento vertical a los elementos hijos; puede ser: flex-start, flex-end, center, strech y baseline

Nota 1: Las dos propiedades - flex-direction y flex-wrap - combinadas, se reducen a una propiedad - flex-flow -, la cual el valor de las dos primeras propiedades, separadas por un espacio, ejemplo:
{
   display:flex;
   flex-flow:column wrap;
}

Nota 2: Align-content determina el espacio entre las líneas, mientras que align-items determina como los elementos en su conjunto están alineados dentro del contenedor. Cuando hay solo una línea, align-content no tiene efecto.
