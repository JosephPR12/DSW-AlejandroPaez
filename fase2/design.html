Propósito del sitio

El propósito del sitio es fomentar el aprendizaje del diseño de sitios web, enfocado en el uso de flexbox, listas y enlaces. Flexbox, empleado para una correcta distribución de los elementos del sitio; listas, usadas para dar orden a los elementos (de forma ordenada o desordenada) a un menú o una enumeración de elementos; enlaces, para dar direccionamiento interno o externo dentro de la misma página o fuera del mismo.

Flexbox

Uso de flexbox: Usado en Hojas de estilo en cascada CSS; es creado para mejorar la forma en que se hace un diseño responsivo del sitio evitando el continuo uso de float, reduciendo la cantidad de lineas de codigo y facilitiando el posicionamiento de los elementos (alterando, alto y distribución de elementos), optimiza el espacio disponible.

La estructura Flexbox, se compone de contendores padre e hijos: El contenedor-Flex es el contenedor padre, contiene las direcciones que definen origen y fin del flujo de elementos hijos que se desean posicionar El elemento-flex es el elemento hijo, útiliza el espacio disponible para ubicarse de acuerdo a las propiedades y ejes (horizontal y vertical) a los cuales está sometido.

Propiedades: Para usar Flexbox se debe iniciar definiendo la propiedad "display: flex" en el contenedor padre:

  • flex-direction: define el eje principal y secundario de elemento hijo, puede ser: row, row-reverse, column, column-reverse
  • flex-wrap: por defecto flex, dispone los elementos en una misma línea, pero si se usa wrap se puede ordenar los elementos en más de una fila o columna; puede ser: nowrap, wrap, wrap-reverse
  • justify-content: da direccionamiento horizontal a los elementos hijos; puede ser: strech, flex-start, flex-end, center, space-between, space-around y space-evenly
  • align-items: da direccionamiento vertical a los elementos hijos; puede ser: flex-start, flex-end, center, strech y baseline
Cabrera G. (2020). Flexbox - ¿Qué es y para qué sirve?. Desde https://somospnt.com/blog/148-flexbox-que-es-y-para-que-sirve



Listas

Uso de las listas: permite crear un conjunto de elementos en forma de lista dentro de la página, precedidos por una viñeta o número

Tipos de listas:

  • Listas ordenadas: muestran los elementos de la lista en orden numérico, alfabético o romano, puede empezar por defecto en número definido usando propiedad start="5", también se puede cambiar el tipo de viñea a alfabética con la propiedad type="A" o por números romanos type="i"
    1. Mariana
    2. Stephani
    3. Melissa
    4. Camilo
  • Listas desordenadas: elementos precedidos por una viñeta (punto, cuadro, asterísco), sin orden
  • Listas de definición: permiten montar listas en las que se tiene una estructura de valor-definición, se constituyen mediante el elemento dl. Dentro del elemento de las listas de definición, tenemos 2 elementos anidados, el valor dt y la definición dd. La estructura queda:
    • Pizpireta
    Que es simpática y de carácter vivaz y alegre
    Pupular
    Moverse de un lado para otro, bullir en algún lugar personas, animales o cosas
    Oxímoron
    Figura retórica de pensamiento que consiste en complementar una palabra con otra que tiene un significado contradictorio u opuesto.

Manual Web (2021). Listas HTML. Desde https://www.manualweb.net/html/listas-html/



Enlaces

Uso de los enlaces: Comunicar una página con otra, para indicar el destino de un enlace se usa el atributo href="" dentro del elemento ancla a, este atributo puede ser cualquier URI que represente un recurso, es decir, una página, una URL genérica, un archivo, un bloque interno a la página.

Destino del enlace: El destino del enlace se especifica con el atributo target (Ej: href="google.com" target="_blank") y son:

  • _blank:abre el enlace en una ventana nueva
  • _self:abre el enlace en la página actual
  • _parent:abre el enlace en el framset superior al que se encuentra la página (el enlace está en el área de frames)
  • _top:abre el enlace en la ventana padre

la etiqueta a (ancla) también ofrece un atributo título (title="") para describir de forma textual el destino del enlace

Manual Web (2021). Enlaces HTML. Desde https://www.manualweb.net/html/enlaces-html/
